# 📊 Superstore Sales Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard built using the Sample Superstore dataset. It provides business insights into sales, profit, customer performance, regional performance, and product categories through interactive visualizations.


---

## 📈 Dashboard Features

- 💰 Total Sales KPI
- 📈 Total Profit KPI
- 👥 Total Customers KPI
- 🛒 Total Orders KPI
- 📊 Profit Margin KPI
- 🌍 Sales by Region
- 📅 Sales Trend by Year
- 📦 Sales by Category
- 👤 Top Customers Analysis
- 📈 Monthly Sales Trend
- 🎯 Interactive Slicers (Year, Region, Category, Segment, Product)

---

## 🛠️ Tools & Technologies

- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
- Microsoft Excel
